GO XML Formats and Tools

Introduction

This project contains descriptions of GO XML formats as well as tools for converting these to other formats.

Background

You should already be familiar with the Gene Ontology project:

  • http://www.geneontology.org
  • http://obo.sourcefore.net
  • GO documentation

Installation

This project lives in the go-dev/xml directory and is part of the GO distribution. This project contains DTDs and XSL stylesheets and requires no installation.

To obtain this project (which is part of the full go-dev distribution), please see the section Software Downloads in the main GO Database and Developers page.

DTDs

For a list of DTDs, see the go-dev/xml/dtd directory.

XSLs

For a list of XSL stylesheets, see the go-dev/xml/xsl directory.

More details can be found on xsl here: http://www.w3.org/Style/XSL/

For XSL processing we use xsltproc, which comes as part of the libxslt distribution

Scripts

The script go-apply-xslt is a simple wrapper for xsltproc. It requires the environment variable GO_ROOT is set to the directory containing the xml/xsl directory
