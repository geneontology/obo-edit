GO-DEV: The GO Software Developers Toolkit

Introduction

GO-DEV is a collection of applications, tools, libraries and schemas developed by the BDGP for dealing with GO and OBO style ontologies.

Background

You should already be familiar with the Gene Ontology project:

  • http://www.geneontology.org
  • http://obo.sourcefore.net

GO Database and Developers page:

  • http://www.godatabase.org/dev

Download

For download information, please read carefully the section "Software Downloads" on the main GO DB Website: http://www.godatabase.org/dev

Installation

GO-DEV is a collection of tools and libraries - please refer to the documentation on each specific subproject (below) for installation. There may be dependencies between the subprojects.

Projects

DAG-Edit
This Java application provides an interface to browse, query and edit GO or any other vocabulary that has a DAG data structure. The most current version of DAG-Edit can be downloaded from the publicly accessible source repository at SourceForge. Help documentation is now included in the program, or is avaliable here. DAG-Edit is independent of and has no dependencies on the other projects in go-dev.
sql
Schema and documentation for the GO relational database. Database downloads available from http://www.godatabase.org/dev/database
go-perl
The go-perl library provides modules to help parse and navigate GO/OBO style ontologies together with associated gene product annotations. It includes an object model and XML-event producing parsers. go-perl is also available from CPAN. go-perl also comes with a number of scripts for converting between ontology file formats and utilities such as map2slim
go-db-perl
The go-db-perl library provides an object oriented API (Application Programmer Interface) for querying the GO Database, as well as scripts for storing data to and retrieving data from the database. go-db-perl requires go-perl
AmiGO
AmiGO is the web browser for the GO Database. The code for AmiGO is distributed as part of the go-dev package. AmiGO requires both go-db-perl and go-perl to run on your server.
XML
XML format specifications and XSL transformations for GO/OBO and other ontology formats such as OWL
Obol
Obol is an experimental research project. It includes a grammar to lexically decompose GO and OBO terms into formal definitions, which can be reasoned over, or exported to OWL. It also allows prolog querying over the GO and OBO ontologies.
