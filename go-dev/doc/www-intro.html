Gene Ontology Software and Databases


Welcome to the GO Software and Database page at the SGD. Here you can find the GO database, as well as software libraries and applications for querying and managing GO and other OBO-style ontologies. For a complete list of tools, please refer to the main GO website
  • comprehensive list of GO tools

DAG Edit

This Java application provides an interface to browse, query and edit GO or any other vocabulary that has a DAG data structure. The most current version of DAG-Edit can be downloaded from the publicly accessible source repository at SourceForge. Help documentation is now included in the program, or is avaliable here.

GO Database

The GO flatfiles and associated data are incorporated into a MySQL database on a monthly basis.

The GO-DEV toolkit

The GO-DEV toolkit is the collection of scripts, applications and database schema modules above developed by the BDGP. It includes the software above, as well as object oriented libraries in perl and java for handling GO and OBO style ontologies.

  • go-dev documentation

Software Downloads

All of the software we develop is open source, and is available from the sourceforge site. DAG Edit has frequent releases. For the rest of the software, you should use CVS to obtain the latest version. If CVS is not an option, you can download the latest monthly release of the code from the GO Database Download Area -- please select the file go_YYYYMM-utilities-src.tar.gz.
