Contructor for the bbop.golr.manager.jquery.browse object.

Yes, that is very long.

This is a specialized (and widgetized) subclass of bbop.golr.manager.jquery.

The functions for the callbacks look like function(term_acc, term_doc){}.  If no function is given, an empty function is used.
