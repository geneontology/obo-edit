Example Scripts

The following example scripts are available in docs/examplescripts:

  • listterms.osl - Prints a list of all the terms in an ontology
  • remove_spaces.osl - Replaces all the spaces in term names with another character (underscores by default)
  • update_gene_associations.osl - Updates a gene association file by updating all references to secondary ids to primary ids, and using replaced_by and consider information to update references to obsolete terms.
  • strip_replacement_tags.osl - Removes all the replaced_by and consider tags in a file.
