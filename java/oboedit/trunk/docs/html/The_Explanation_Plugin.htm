Explanations

The Explanations panel displays the explanation for any currently selected links. When the Explanations panel is opened, it looks like this:

The Explanations panel contains a plain text explanation of why the selected link exists. The explanation may contain a number of hyperlinks. If a term name or link description is hyperlinked, clicking the link will cause the term or link to be selected in the primary ontology editor.

Sometimes an explanation for an implied link will cite other links that were used as evidence to create the implied link. These evidence links may be followed by a bold hyperlink titled show explanation. If a show explanation link is clicked, the explanation for the associated link will appear beneath the link, and the link text will change to hide explanation. The show/hide explanation links can be used to follow the chains of implications the reasoner often uses to create an implied link.

At the bottom of the Explanations panel is a checkbox labaled Show verbose explanations. If this checkbox is selected, additional text may be displayed clarifying how a particular reasoner rule works. Experienced users may decide to turn off this additional text to simplify the display, especially when a lot of sub-explanations are also being displayed.
