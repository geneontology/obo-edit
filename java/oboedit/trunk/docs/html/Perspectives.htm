Perspectives

Layouts in OBO-Edit are referred to as Perspectives. OBO-Edit offers several built-in perspectives and also allows users to customize their own perspectives (which can then be saved and shared with other users). To access the menus options for perspectives, select [View]->[Layouts].

Creating Your Own Perspective

In addition to these built-in perspectives you can create your own favorite perspective, save it, and (if desired) share the resulting configuration file with other people. You can set up your desired perspective by using the [View]->[Show] menus to add components, and/or the "X" icons to remove components from the display.

Components can be moved around by grabbing their title bars and dragging them around. As you drag a panel, you will see a black outline that shows where it will land if you drop it. You can turn a panel into an independent window by clicking the icon of an arrow that points upwards and to the right:

To return a panel to the main OBO-Edit window, click its icon that shows an arrow pointing downwards and to the left.

Saving and Sharing Perspectives

First set up your perspective by adding or removing panels and dragging them to your preferred locations. Then save the perspective (using [View]->[Layouts]->[Save current perspective as..]). (Your perspective will now show up in the list of perspectives when you access [View]->[Layouts].)

To find your perspectives file to share with other users, go to your .oboedit directory (current this is .oboeditbeta but it won't be once the tool is released) and look inside the perspectives directory for the appropriate .idw file. It is not possible to edit this file by hand, but you can mail the file around to let others use your favorite perspective. They simply need to add your file to their .oboedit/perspectives directory.
