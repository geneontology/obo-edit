User Defined Checks

User defined checks are checks that a user creates. They display errors or warnings based on whether a user-specified filter is matched. See The Verification Manager for information on how to create a user-defined check.

  • Namespace Check
  • is_a complete Check

Namespace Check

The user might be interested in checking if all terms have the correct namespace setting. It is possible to create a user-defined Namespace Check and a warning will appear if a term has, for example, 'gene_ontology' as its namespace.

Note: The Part_of relation in Gene Ontology has default namespace equal to 'gene_ontology'.
