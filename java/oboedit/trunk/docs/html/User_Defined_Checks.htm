User Defined Checks

User-defined checks are checks that a user creates. They display errors or warnings based on whether a user-specified filter is matched. See The Verification Manager for information on how to create a user-defined check.

  • Namespace Check
  • is_a complete Check

Namespace Check

The user might want to check whether all terms have the correct namespace setting. It is possible to create a user-defined Namespace Check to give a warning if a term has the wrong namespace. For example, in the Gene Ontology, only the relations should have 'gene_ontology' as the namespace. This check warns if any classes have the namespace 'gene_ontology
