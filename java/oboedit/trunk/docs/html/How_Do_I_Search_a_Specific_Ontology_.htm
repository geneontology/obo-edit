How Do I Search a Specific Ontology?

 

Load and visualize ChEBI ontology

You might have trouble loading and visualizing the ChEBI ontology in OBO-Edit. OBO-Edit might show the ontology in an unhelpful "flat" form like this:

To solve the problem, you need to select "View" -> "Show" -> "Config" -> "Global Filters". Then select "Link filters" and delete the second filter: Select links where Type that have a Is Transitive.

Once you have saved the changes, you need to restart OBO-Edit. After you reload the ChEBI ontology, it should display correctly:
