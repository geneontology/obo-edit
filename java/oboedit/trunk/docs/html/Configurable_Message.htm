Configurable Message

 

The Configurable Message feature can be used to add a static message to the interface to help specific user groups. For example, if you are asking a group of domain experts to review your ontology changes then you can enter a static message that will help them to find the terms of interest.

To add your own message, start the Configurable Message tab and click the the little wrench (or, if you're British, the spanner)--the leftmost of the icons in the top bar of the tab. Enter your message using html to format it.

Then click the checkmark or tickmark icon to save the message.

To ensure that the layout appears as you require it and displays your message you will need to make a special perspective for your users. [See save and load perspectives]
