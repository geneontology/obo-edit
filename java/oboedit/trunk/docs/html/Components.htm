Components

 

OBO-Edit 2 has multiple components that can be added to the display using the "Show" menu. The components can be dragged around to rearrange them, or even detached from the main OBO-Edit window as free-floating windows. See the section about Perspectives for more information about configuring your view.

About the little icons on each component

Most components have a "?" icon. If you click this, it will launch a window with help for that specific component.
