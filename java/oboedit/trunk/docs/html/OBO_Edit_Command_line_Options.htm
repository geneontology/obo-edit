OBO-Edit Command-line Options

OBO-Edit supports the following command-line options:

  • -verbose - Displays verbose status messages while OBO-Edit is running
  • --listadapters - Lists all the available data adapters and exits
  • -load <adapter name> <file name> (default) Loads a file on startup. This parameter is the default parameter, meaning that it is implicit, and does not need to be specified. If no adapter name is provided, -OBO_EDIT:OBO_Adapter is assumed. Hence, if you want to load an OBO file, all you need to provide is the file name with no other arguments. By default, the following adapters are available:
    • -OBO_Edit:OBO_Adapter - The OBO Adapter
    • -OBOEDIT:GOFlatFileAdapter - The GO Flat File Adapter
    • -OBOEDIT:Serial - The Serial Adapter
