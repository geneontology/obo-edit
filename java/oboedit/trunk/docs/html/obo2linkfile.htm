obo2linkfile

The obo2linkfile utility takes an OBO file and produces a tab-delimited text file that contains the transitive closure for the ontology in the OBO file.

obo2linkfile <obo_file> -o <output_file>

Parameters:

  • <obo_file> - Input file.
  • -o <output_file> - Output file.

Example:

Make the transitive closure text file for "foo.obo". A proper CLASSPATH has been added.

CLASSPATH=/home/sjcarbon/tmp/obo.jar:/home/sjcarbon/local/src/svn/geneontology/java/oboedit/trunk/classes:/home/sjcarbon/local/src/svn/geneontology/go-dev/gwt/external/bbop.jar:/usr/share/java/log4j-1.2.jar /home/sjcarbon/local/src/svn/geneontology/java/oboedit/trunk/launch_scripts/obo2linkfile foo.obo -o link.txt
