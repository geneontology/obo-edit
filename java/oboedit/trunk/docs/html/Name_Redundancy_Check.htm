Name Redundancy Check

This check ensures that term names in the ontology are unique, and issues a warning if they are not. Note that name uniqueness is not required, but it's a good idea.

This check displays a warning if:

  • Two terms have the same primary name
  • A term name is the same as an exact synonym of some ontology term
  • An exact term synonym matches the name of some ontology term
  • Two terms have the same exact synonym

Note that a redundant name will always cause at least 2 warnings to be generated, because a warning will appear for each term that shares the name or synonym.

If two terms have the same synonym, but the synonym scope is not "exact" for both terms, a warning is not generated. This is a perfectly legal condition that can arise from a term split or community usage conventions.
