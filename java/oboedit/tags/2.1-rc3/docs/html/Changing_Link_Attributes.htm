Changing Link Attributes

Works by: Right-click, edit menu

Hot keys: None

Source requirements: One or more source relationships required

Target requirement: No target required

It is possible to use the ontology editor panel to change the necessity, inverse necessity, and intersection status (completeness) of a relationship.

To change any of these attributes, select one or more source relationships, then open the right-click or edit menu. There will be menu items for editing these link attributes:

To change the current value of one of these attributes, select one of these menu items. A flyout menu will appear:

The current value of the attribute will be grayed out. To change the attribute value, choose the appropriate menu item.

It is possible to change an attribute value for many source relationships at once. The attribute value will be changed for every selected source.
