The <searchbutton> Tag

The <search_button> tag creates a custom search button. See Custom Search Buttons for more information about how these versatile components work.

Parameters

Name Description Legal Values Default
method

Specifies the type of search button this is.

SEARCH

FILTER

RENDER

MODIFY

SEARCH

filter

Specifies a filter for this button to use. The value is either a path to a saved filter file on the disk, or a built-in filter specifier. This parameter must be specified unless the method paramter is set to MODIFY. If a MODIFY button is selected with no filter set, no modification filter will be applied.

any filter path or a valid built-in filter specifier

none

label The text label that appears on the button. This may be left blank if the icon parameter is defined. any string none
icon An icon that should appear on the button. The value is either a path to a .gif file on disk. This may be left blank if the label parameter is defined.

the path to a .gif file on disk

 none
tooltip Tooltip text that will appear when the mouse hovers over this button. This parameter is optional. any string none
selected Whether or not this button is selected by default. This parameter only makes sense if the method parameter is set to MODIFY.

true

false

 false
buttonGroup An identifier for this button's buttonGroup. It can be any string, but if multiple buttons have the same buttonGroup, only one button in that group can be selected at a time. Whenever a button in the buttonGroup is selected, all other buttons in the group are deselected.

any string
