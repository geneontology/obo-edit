Custom Data Adapters

At present, OBO-Edit is packaged with only three data adapters: the OBO Adapter, the GO Flat File Adapter, and the Serial Adapter. However, data adapters are pluggable components. Third-party developers can create and distribute new data adapters for new data types. These custom data adapters may provide access to a SQL database, a description logic language, or some proprietary, in-house file format.

When using a third party data adapter, please refer to the developers instructions for information on how to install and use the data adapter.
