The Dbxref Library Plugin

The Dbxref Library Plugin allows users to keep a database of frequently used dbxrefs to save the effort of typing long or obscure dbxrefs each time they are needed.

Using Dbxrefs in the Library

To use a dbxref from the library, select the dbxref from the list. Drag the dbxref to any dbxref list in the  text editor or press the "Use dbxref" or "Use as def dbxref" button. Drag and drop must be used to assign a Dbxref Library dbxref to a synonym.

Modifying the Library

The easiest way to add dbxrefs to the library is to drag them from any dbxref list in the text editor onto the Dbxref Library.

To add new dbxrefs, or to remove items from the list, press the "Configure dbxrefs" button. The Dbxref Library configuration window will appear:

This window contains a dbxref editor list. This list acts like any other dbxref editor list. See Text Editing for more information. Press "Save configuration" to save your changes.

Sharing Your Library

Dbxref libraries can be saved to an external file. Dbxref lists can be imported into other installations of OBO-Edit.

To export your dbxref list to a file, press the "Export dbxrefs" button. A file browser window will appear where a save file can be specified.

To import a dbxref list from a file, press the "Import dbxrefs" button. A file browser window will appear where a file can be chosen off the disk. Beware! The imported dbxrefs will replace the existing dbxref library.
