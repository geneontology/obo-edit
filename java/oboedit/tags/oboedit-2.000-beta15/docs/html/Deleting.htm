Deleting

Works by: Edit menu, right-click

Hot keys: Del

Source requirements: One or more source relationships required

Target requirement: No target required

The Delete operation removes a relationship from the ontology. When a term's last parent relationship is Deleted, the term becomes obsolete. If a term has children, it cannot be deleted; the children must be removed first.

To delete one or more relationships, first select the relationships to be deleted. Choose the "Delete" option from the edit menu or right-click menu. The relationships will be removed, and - if appropriate - orphaned terms will be marked obsolete.

The Delete operation can also be used to make a root term obsolete. To obsolete a root term, select the term, and choose "Delete" from the right-click or edit menu.
