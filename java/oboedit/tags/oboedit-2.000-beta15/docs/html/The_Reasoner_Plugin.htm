The Reasoner Plugin

The reasoner is activated and deactivated via the reasoner plugin. If the reasoner is inactive, the plugin looks like this:

To activate the reasoner, click the "Use reasoner" button.

When this button is pressed, the reasoner starts up and begins reasoning. The progress bar may fill up several times before reasoning is complete; there is no way to predict in advance how many reasoner passes will be necessary to reason over an ontology, so the progress bar simply shows the progress of the current reasoner pass.

Once reasoning is complete, a status message appears:

The first line gives the number of milliseconds elapsed during reasoning.

The second line shows how many implied links were discovered.

The third line shows how many implied links are actually shown. Most implied links are trimmed (see trimming). This line shows how many implied links were not trimmed.

The final line shows how many redundant links were found in the ontology.

To turn off the reasoner, unclick the "Use reasoner" box.

Wait... Did I Leave the Reasoner On?

Like any other plugin, the reasoner plugin remembers its state when OBO-Edit is exited. This means that if the reasoner is on when you exit OBO-Edit, it will be on the next time you start OBO-Edit.

This entire application runs more slowly when the reasoner is running, so enable it with caution.
