Organizing Your Ontology: Editing Metadata

Metadata are data about data. In the context of ontologies, metadata describe properties of the ontology overall, rather than properties of individual terms or relationships in the ontology. Ontology metadata could include information such as the URL where the official version of the ontology lives; the date it was last updated; a contact person for the ontology. In OBO-Edit, the types of metadata that can be viewed and edited are:

  • Subsets to which terms can be assigned
  • ID generation profiles
  • Namespaces
  • Synonym types
  • Obsolete terms or relations

To access these panels, use the Metadata menu items.
