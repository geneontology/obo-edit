obo2linkfile

The obo2linkfile utility takes an OBO file and produces a tab-delimited text file that contains the transitive closure for the ontology in the OBO file.

obo2linkfile <obo_file> -o <output_file>

The command line parameters are:

  • <obo_file> - Input file.
  • -o <output_file> - Output file.

Example:

Make the transitive closure text file for "foo.obo". Set CLASSPATH to include obo.jar, bbop.jar and log4j-1.2.15.jar present in the runtime directory.

CLASSPATH= /home/user/local/OBO-Edit2b52/runtime/obo.jar:/home/user/local/OBO-Edit2b52/runtime/bbop.jar:/home/user/local/OBO-Edit2b52/runtime/log4j-1.2.15.jar obo2linkfile foo.obo -o link.txt
